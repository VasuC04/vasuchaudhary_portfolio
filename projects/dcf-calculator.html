Why Build This in Python?

The typical DCF model lives in Excel — powerful, but slow to update, error-prone across linked sheets, and impossible to run at scale. This project rebuilds the same logic in Python, making the valuation engine modular, repeatable, and extendable.

The goal is a tool that takes a company's financial inputs and outputs a clean valuation range in seconds — with sensitivity tables and scenario outputs generated automatically, not manually recalculated every time.

Practical Application

In an M&A context, speed matters. Being able to run a DCF on a target company quickly — with scenario toggles and sensitivity outputs ready to paste into a pitch deck — is a genuine workflow advantage over pure Excel modelling.

The Python script also exports a formatted Excel file via openpyxl, meaning the output is immediately presentable and compatible with standard IB workflows.

04 — Code Structure

How the engine is built

01
Config Input Layer — All financial assumptions (growth rates, margins, WACC inputs, shares outstanding) live in a single Python dictionary. Switching company requires changing one block.
02
WACC Module — A standalone function takes cost of equity inputs (Rf, beta, ERP) and cost of debt inputs, returning a single WACC float used downstream in discounting.
03
FCF Projection Module — Projects revenue, EBITDA, D&A, capex, and working capital changes over 5 years, returning a Pandas DataFrame of annual unlevered free cash flows.
04
Valuation Module — Discounts FCFs at WACC, computes terminal value via perpetuity growth formula, and bridges enterprise value to equity value per share.
05
Sensitivity Engine — Runs the valuation module iteratively across a WACC × terminal growth rate grid using NumPy, producing a clean sensitivity matrix as a DataFrame.
06
Excel Export — Writes all outputs to a formatted .xlsx workbook via openpyxl — tabs for assumptions, FCFs, valuation summary, and colour-coded sensitivity table.
# DCF Calculator — core valuation function

def calculate_dcf(fcf_list, wacc, terminal_growth, net_debt, shares_out):
    # Discount each year's FCF to present value
    pv_fcfs = [fcf / (1 + wacc) ** (i + 1) for i, fcf in enumerate(fcf_list)]

    # Terminal value via Gordon Growth Model
    terminal_value = fcf_list[-1] * (1 + terminal_growth) / (wacc - terminal_growth)
    pv_terminal = terminal_value / (1 + wacc) ** len(fcf_list)

    # Enterprise value → equity value per share
    enterprise_value = sum(pv_fcfs) + pv_terminal
    equity_value = enterprise_value - net_debt
    return equity_value / shares_out
